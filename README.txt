1) Открой Qt Creator -> Open File or Project -> выбери EmergencyPanel.pro
2) Убедись, что у тебя установлены модули Qt Quick и Qt Quick Controls 2.
3) Поставь в папку проекта файл logo.png (логотип).
4) Сборка / Запуск.

Файлы:
- main.qml  — UI (QML)
- LineItem.qml — компонент строки списка
- main.cpp — стандартный загрузчик QML
- resources.qrc — ресурсы (логотип)

Дальше:
- Можно заменить ListModel на модель с C++ (QAbstractListModel), если нужна реальная интеграция с оборудованием.
- Добавить подтверждающие диалоги перед переводом в тест/сброс.
- Реализовать websocket/serial/Modbus интеграцию в C++/Python backend по необходимости.
Философия: У тебя заводской шкаф на определенное количество линий
Линии все прогятся на заводе через json
Если линия не нужна, выключаем ее через состояни notUsed
